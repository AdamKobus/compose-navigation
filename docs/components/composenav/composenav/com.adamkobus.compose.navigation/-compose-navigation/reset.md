# reset

[androidJvm]\
fun [reset](reset.md)()

Resets the ComposeNavigation to initial state. This will reset:

- 
   logger
- 
   navigation actions verifiers
- 
   navigation intent resolvers
- 
   main dispatcher
- 
   io dispatcher
- 
   reserved names flag (it will become enabled again)
- 
   navigation processing timeout
